# ExplanationMode Component 가이드

## 디자인 의도
초등학생 눈높이에 맞춘 인물 설명 모드 전용 컴포넌트입니다. 사용자가 27명의 인물에 대해 학습할 수 있도록 크게 세 부분으로 나누어 직관적인 레이아웃을 제공합니다. 

- 좌측 카드: 화면의 약 40% (lg:col-span-5)를 차지하며 인물의 이미지를 크게 보여주고 이름 레이블을 씌워 누군지 명확히 인식하도록 합니다.
- 우측 상단 카드: 인물이 가진 '핵심 가치'를 스타 아이콘, 그라데이션, 큰 텍스트 등 시각적으로 강조하여 직관적으로 전달합니다.
- 우측 하단 카드: 인물의 '설명(description)'을 카드 형식으로 노출하며, 글씨 크기를 키우고 여유로운 간격으로 배치해 가독성을 높입니다. 

전반적인 배경은 다른 페이지와 동일하게 어두운 테마(`bg-[#0a0a0a]`)에 미세한 보라색 Glow를 추가해 집중도를 높입니다.

## Props 구조
```tsx
interface ExplanationModeProps {
  onExit: () => void; // 게임이나 설명 모드를 종료하고 메인 랜딩으로 돌아갈 때 호출
}
```

## 구현 방식
- **상태 관리**: 
  - `currentIndex`: 현재 화면에 띄울 인물의 배열 내 순서 번호. (초기 0) 좌하단에 위치한 '이전', '다음' 화살표 버튼을 통해 상태값을 변경합니다.
- **애니메이션**: 
  - `framer-motion`의 `AnimatePresence`와 `motion.div`를 사용하여 카드 변환 시 부드러운 페이드 인/아웃 및 스케일 효과를 줍니다. 
- **설명 데이터 모델**:
  - `characters.ts` 모델의 `coreValue`, `elementaryDescription` 컬럼 데이터를 이용해 렌더링합니다. 설명 텍스트는 길이가 짧은 편이므로 특별히 스크롤바는 필요치 않으며 자동 줄바꿈을 활용합니다.
- **레이아웃**:
  - 가장 상위 요소를 Flexbox 중앙 정렬로 두고 화면의 100%를 쓰지 않고 카드를 `Grid` 시스템에 가둡니다. 모바일 환경에선 `lg` 브레이크포인트를 기준으로 세로 배치 형식으로 유연하게 해체되도록 구조를 잡았습니다.
